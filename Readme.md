# OverSwim — Landing Page (MG Studios Web)

Landing page moderna y responsiva para **OverSwim** (clases de natación).  
Diseñada con una estética fresca tipo “agua premium”, usando una paleta definida y tipografías elegantes.

**Co-fundador (MG Studios Web): Grar2** 

---

## ✨ Características

- ✅ Diseño **responsive** (móvil, tablet y desktop)
- ✅ Paleta de colores de marca:
  - **#FAA916** (acento / call-to-action)
  - **#A6E1FA** (fondo / agua)
  - **#01295F** (texto / azul profundo)
- ✅ Tipografías:
  - **Galada** (principal: títulos / branding)
  - **Montserrat** (secundaria: texto general)
- ✅ Secciones típicas de landing:
  - Hero (mensaje principal)
  - Servicios / clases
  - Beneficios
  - Testimonios
  - Contacto / CTA (WhatsApp u otros)
- ✅ Fácil de personalizar: logo, imágenes, textos, links
